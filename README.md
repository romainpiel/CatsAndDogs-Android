# Cats And Dogs Talk: Kotlin Android App
This is the Kotlin (for Android) version of the cats and dogs app example code. It connects to the [backend for frontend](https://github.com/RomainPiel/CatsAndDogs-kotlin-bff) to get it's view model data to render.
